本文档目录包含教程的详细说明、练习题和参考资料。

```
docs/
```

## 📖 使用指南

### 初学者路线

1. 阅读主 README.md 了解教程概况
2. 按照 `01-入门阶段/` → `02-进阶阶段/` → `03-高级阶段/` → `04-骨灰阶段/` 顺序学习
3. 每章学习后完成 `exercises/` 中的练习题
4. 遇到问题查看 `知识索引.md`

### 快速查找

- 需要编译帮助 → `编译指南.md`
- 查找某个知识点 → `知识索引.md`
- 做练习题 → `exercises/`
- 查看总结 → `项目总结.md`

## 🎯 学习建议

1. **循序渐进** - 按顺序学习，不要跳章
2. **动手实践** - 每个示例都要亲手运行
3. **完成练习** - 练习题是巩固知识的关键
4. **善用索引** - 遇到问题先查知识索引
