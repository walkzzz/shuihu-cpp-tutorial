# 🏮 水浒聚义寨 C++ 教程 - 编译指南

## 📋 环境要求

### Windows

**选项 1：Visual Studio 2022（推荐）**
- 安装 Visual Studio Community 2022
- 安装"C++ 桌面开发"工作负载

**选项 2：MinGW-w64**
- 下载：https://www.mingw-w64.org/
- 添加到 PATH 环境变量

**选项 3：WSL2（Windows Subsystem for Linux）**
- 安装 WSL2 和 Ubuntu
- `sudo apt install g++ cmake`

### macOS
```bash
xcode-select --install
brew install cmake
```

### Linux
```bash
# Ubuntu/Debian
sudo apt install g++ cmake make

# Fedora/RHEL
sudo dnf install gcc-c++ cmake make
```

## 🚀 编译方法

### 方法 1：使用 CMake（推荐）

```bash
# 1. 创建构建目录
mkdir build && cd build

# 2. 配置
cmake ..

# 3. 构建
cmake --build . --config Release

# 4. 运行主程序
./shuihu-cpp-tutorial  # Linux/macOS
.\Release\shuihu-cpp-tutorial.exe  # Windows

# 5. 运行示例
cmake --build . --target basic-01
./basic-01
```

### 方法 2：使用 g++ 直接编译

```bash
# 编译单个示例
g++ -std=c++17 -o basic-01 examples/basic/01-环境搭建.cpp
./basic-01

# 编译主程序
g++ -std=c++17 -o main src/main.cpp
./main
```

### 方法 3：使用 MSVC（Windows）

```powershell
# 打开"Developer Command Prompt for VS 2022"

# 配置
cmake -G "Visual Studio 17 2022" -B build

# 构建
cmake --build build --config Release

# 运行
.\build\Release\basic-01.exe
```

## ⚠️ 常见问题

### Q1: CMake 找不到编译器

**解决：**
- Windows: 安装 Visual Studio 2022
- macOS: 运行 `xcode-select --install`
- Linux: 安装 `build-essential` 包

### Q2: 中文文件名编译错误

某些编译器可能不支持中文文件名。解决方法：

1. **重命名文件为英文**
2. **使用 WSL2（Linux 子系统）**
3. **使用 PowerShell 直接编译**

### Q3: C++17 不支持

确保编译器支持 C++17：
```bash
g++ --version  # 需要 GCC 7+
clang++ --version  # 需要 Clang 4+
```
