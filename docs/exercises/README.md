## 📋 练习题说明

每章练习题分为三个难度：

- 🟢 **基础题** - 巩固知识点，必做
- 🟡 **提升题** - 挑战自我，选做
- 🔴 **实战题** - 综合应用，挑战题

## 📘 入门阶段练习题

### 第 1 章：环境搭建

#### 🟢 基础题

1. 安装 C++ 编译器（GCC/Clang/MSVC）
2. 验证安装：`g++ --version` 或 `cl`
3. 安装 CMake（3.15+）
4. 克隆教程仓库并成功编译 basic-01

#### 🟡 提升题

1. 配置 VS Code 的 C++ 开发环境
2. 尝试使用不同的编译器编译同一程序
3. 比较不同编译器的编译速度和输出文件大小

#### 🔴 实战题

1. 编写一个脚本自动完成环境配置
2. 在 GitHub 上创建自己的仓库，上传编译好的程序

### 第 2 章：第一个程序

#### 🟢 基础题

1. 修改示例，输出自己的名字和绰号
2. 添加一个新的头领点将
3. 修改输出格式，使其更美观

```cpp
// 示例：添加自己的头领
点将 ("你的绰号", "你的名字", 你的兵力);
```

#### 🟡 提升题

1. 创建一个函数，接受三个参数并输出
2. 使用循环输出 10 个头领
3. 尝试使用不同的输出格式（endl, \n, \t）

#### 🔴 实战题

1. 编写一个完整的"聚义厅点将"程序，包含 108 将
2. 使用数组或 vector 存储头领信息
3. 添加用户交互，让用户输入头领信息

### 第 3 章：变量与数据类型

#### 🟢 基础题

1. 定义以下变量并输出：
   - 你的姓名（string）
   - 你的年龄（int）
   - 你的身高（double）
   - 是否好汉（bool）

2. 计算并输出：
   - 马军、步军、水军的总兵力
   - 粮草的平均分配量

3. 使用 const 定义圆周率π=3.14159

#### 🟡 提升题

1. 尝试不同类型之间的转换（隐式和显式）
2. 使用 auto 关键字让编译器推断类型
3. 计算并输出各军兵种占总兵力的百分比

```cpp
// 示例
auto 总兵力 = 马军 + 步军 + 水军;
double 马军比例 = static_cast<double>(马军) / 总兵力 * 100;
```

#### 🔴 实战题

1. 编写"梁山泊兵马统计系统"
   - 输入各军兵力
   - 计算总兵力
   - 计算各军比例
   - 输出统计报告

2. 使用 struct 组织头领数据：
```cpp
struct 头领 {
    std::string 姓名;
    std::string 绰号;
    int 兵力;
    double 武力值;
};
```

```cpp
// 默认参数示例
void 点将 (std::string 姓名，int 兵力 = 1000);
```

#### 🔴 实战题
