## 🔤 按字母顺序

### A
- **atomic (原子操作)** - 骨灰阶段 - 并发编程基础
- **advanced (高级阶段)** - 智能指针、容器、Lambda、异常、移动语义

### B
- **bool (布尔类型)** - 入门阶段 - 真/假值
- **basic (入门阶段)** - 环境、变量、函数、引用

### C
- **class (类)** - 进阶阶段 - 面向对象基础
- **const (常量)** - 入门阶段 - 不可变的值
- **constexpr (常量表达式)** - 高级阶段 - 编译期计算
- **container (容器)** - 高级阶段 - vector, map, set 等

### D
- **double (双精度浮点数)** - 入门阶段 - 小数类型

### E
- **exception (异常)** - 高级阶段 - 错误处理机制

### F
- **for 循环** - 入门阶段 - 迭代结构
- **function (函数)** - 入门阶段 - 代码复用

### I
- **if 条件** - 入门阶段 - 分支结构
- **int (整数)** - 入门阶段 - 整型数据
- **inheritance (继承)** - 进阶阶段 - 类之间的关系
- **intermediate (进阶阶段)** - 面向对象、类、继承、模板

### L
- **Lambda 表达式** - 高级阶段 - 匿名函数
- **long (长整数)** - 入门阶段 - 更大范围的整数

### M
- **map (映射)** - 高级阶段 - 键值对容器
- **master (骨灰阶段)** - 多线程、并发、内存管理、设计模式
- **move semantics (移动语义)** - 高级阶段 - 高效资源转移

### O
- **operator (运算符)** - 进阶阶段 - 运算符重载
- **ostream (输出流)** - 入门阶段 - cout 相关

### P
- **pointer (指针)** - 高级阶段 - 内存地址
- **polymorphism (多态)** - 进阶阶段 - 多种形态

### R
- **reference (引用)** - 入门阶段 - 变量别名
- **return (返回)** - 入门阶段 - 函数返回值

### S
- **set (集合)** - 高级阶段 - 不重复元素容器
- **shared_ptr (共享指针)** - 高级阶段 - 共享所有权
- **smart pointer (智能指针)** - 高级阶段 - 自动内存管理
- **string (字符串)** - 入门阶段 - 文本数据
- **struct (结构体)** - 进阶阶段 - 数据聚合

### T
- **template (模板)** - 进阶阶段 - 泛型编程
- **thread (线程)** - 骨灰阶段 - 并发执行
- **type (类型)** - 入门阶段 - 数据类型

### U
- **unique_ptr (独占指针)** - 高级阶段 - 独占所有权
- **unsafe (不安全)** - 骨灰阶段 - 底层操作

### V
- **vector (向量)** - 高级阶段 - 动态数组
- **virtual (虚函数)** - 进阶阶段 - 多态基础

### W
- **weak_ptr (弱指针)** - 高级阶段 - 弱引用

## 🔍 快速查找代码示例

```cpp
// 变量定义
int 兵力 = 5000;
const std::string 寨主 = "宋江";

// 函数定义
void 点将 (const std::string& 姓名) {
    std::cout << 姓名 << " 听令!" << std::endl;
}

// 类定义
class 头领 {
public:
    头领 (std::string n) : 姓名 (n) {}
    void 点将 () { std::cout << 姓名 << " 听令!" << std::endl; }
private:
    std::string 姓名;
};

// 智能指针
auto 林冲 = std::make_unique<头领>("林冲");
auto 鲁智深 = std::make_shared<头领>("鲁智深");

// Lambda 表达式
auto 增兵 = [](int b, int 增量) { return b + 增量; };

// 模板
template<typename T>
T 最大值 (T a, T b) { return (a > b) ? a : b; }
```
